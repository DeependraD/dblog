## Edit, tabbed section and referencing for html::document

If you want to add edit button, tabbed sections, and text referencing features all on single page `rmarkdown::html_document`, follow this post: https://yongfu.name/2019/02/10/rmd_edit_btn.html. Note this is **not** intended for `blogdown::html_page`.
